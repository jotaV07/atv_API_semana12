## Endpoint Utilizado
- **Filmes Populares:** `GET /movie/popular` — carregado automaticamente ao abrir a página
- **Busca por Nome:** `GET /search/movie?query=...` — ativado ao digitar e clicar em "Buscar"

## Fluxo da Aplicação
1. **Requisição:** Ao carregar a página, a função `init()` chama `fetchMovies()` que faz um `fetch` assíncrono com `async/await` para o endpoint de filmes populares da API do TMDB, passando a API Key e o idioma `pt-BR`.
2. **Tratamento:** A resposta é convertida para JSON e o array `results` é extraído. Em caso de erro HTTP ou falha de rede, uma mensagem de erro é exibida ao usuário.
3. **Renderização:** A função `renderMovies()` limpa o container e chama `createMovieCard()` para cada filme, criando elementos HTML dinamicamente com `createElement` e `appendChild`, exibindo poster, título, ano e nota.

## Funcionalidades
- Listagem automática de filmes populares ao abrir a página
- Busca por nome de filme em tempo real
- Cards com poster, título, ano de lançamento e nota média
- Badge de nota colorido (verde/amarelo/vermelho) conforme avaliação
- Mensagem de estado (carregando, sem resultados, erro)
- Layout responsivo para mobile
